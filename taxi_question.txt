After the Online Qualification of JobStreet.com Code Challenge 2015, there are N groups of contestants qualified for the Grand Final at UNITAR. There are G[i] people in a group (1 ≤ G[i] ≤ 10) and they all decided to get there by taxi. Each taxi can carry at most 4 people so they decided to split members to minimize cost.

What is the minimum number of taxi required to drive all of them to UNITAR?

Input

Input start with an integer N (1 ≤ N ≤ 10000) is the number of groups. The second line contains N integers G[1], G[2], ..., G[n] (1 ≤ G[i] ≤ 10) is the number of people for each group separated by one space character.

Output

Print a single number — the minimum number of taxi required to drive all of them to UNITAR.

Sample Input

8
2 3 4 4 2 1 3 1
Sample Output

5
